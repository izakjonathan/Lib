# Studio UI v0.3.0

A reusable Next.js component-library browser for GitHub and Vercel.

## Included
- Searchable component catalogue
- Category filtering
- Dedicated component documentation routes
- Desktop, tablet and mobile preview controls
- Copy-to-clipboard usage snippets
- Existing responsive shell and dark/light theme
- Seven reusable UI components

## Run
```bash
npm install
npm run dev
```

## Validate
```bash
npm run lint
npm run build
```

Node.js 24 or newer is expected.
